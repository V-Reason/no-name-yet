# 《逆流而下》
> CiGA GameJam 2026

## 简介

拒绝随波逐流，找到心灵锚点！

《逆流而下》是一款画风可爱的横版深海冒险闯关游戏。在无拘无束却又暗流涌动的海底，浮力会持续将你向上推，你需要借助散布着的远古锚点和铁链，不断向下攀附、甩荡，对抗浮力，向着神秘的百米深海进发。深海中的暗流和沙丁鱼也许会成为你的阻碍，也有可能助你一臂之力。

**这款海底捞游戏真的很香……不是，很萌！**

---

## 灵感来源

游戏的核心灵感，源于深海锚点这一沉默却充满力量的独特意象。在暗流翻涌的海底世界里，锚是为数不多的“恒定之物”——它无声沉落、扎根深处，不随洋流摆动。我们希望把这份安全感融入游戏体验中：当你身处无依无靠的深海，锚点与链条就是你最可靠的依托。每一次抓取锚点、甩出钩子、扣紧链条的瞬间，你都能获得十足的安全感——不必担心被水流冲散或鱼群撞击而失去方向，锚点能让你从容地向着更深处“逆流而下”。

游戏的另一个灵感来源是本队丰富的海产元素（雾）。

---

## 视觉风格

超绝大杂烩风，神秘的深海背景搭配可爱有趣的人物立绘及海洋生物彩蛋。
